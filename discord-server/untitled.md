# 客服支援

此工具能夠為你們解答問題和解決疑惑，請適當小心利用，本群管理員及其他協助人員能夠協助你，但將會拒絕一切不關乎於本群的內容及協助請求。

{% hint style="success" %}
目前支援時間由早上10:00至晚上9:30。最遲會在兩小時內回覆。
{% endhint %}

![](../.gitbook/assets/img_7210.jpeg)

{% hint style="danger" %}
警告: 在非必要的情況下使用此服務將會被警告。  
屢次不聽將會禁止使用此服務。
{% endhint %}

### 受支援協助內容包括

* 身份組事宜
* 伺服器或頻道獎勵事宜
* Youtube會員購買事項
* 用戶紛爭及舉報個別違規內容
* 13歲以下用戶上訴證據內容
* 重大事宜 \(關乎群組安全/緊急事件\)
* 權限錯誤或者漏洞上報
* 或者其他需要動用數位管理員處理的事情

### 但不接受內容包括:

* 你的私人或者其他Discord伺服器
* 你的YouTube頻道問題
* 外部用戶紛爭\(無關本群\)
* 你的私人生活問題
* 請求技術\(群組功能、機器人等\)教學
* 或者其他完全不關乎本群的內容
