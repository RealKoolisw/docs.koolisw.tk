# 規則 Rules

![](../../.gitbook/assets/image0%20%281%29.png)

歡迎來到 Koolisw Army的官方Discord伺服器

這個是由 **超級酷斯RealKoolisw** 創建的官方Discord伺服器, 於2020年3月11日創立, 並基於Draconian Workshop以及Koolisw的YouTube頻道所持有的  
  
在你加入伺服器的時候, 即代表你同意以及必須遵守所有群組規則並且遵守管理人員的規定

> Koolisw Nation伺服器詳細規則 於2020年9月9日修訂

 **\[1\] 請遵守本伺服器的所有成員** 必須保持友善, 並不可以對任何成員造成困擾 例如: 頻繁地ping某位用戶/恐嚇/任何形式的騷擾, 我們將會已對該用戶的困擾程度作出懲罰  
  
 **\[2\] 禁止洗版** 例如: 文字/表情符號/影片/任何形式。 若果不斷地發出無關聊天內容的信息, 會被靜音作為懲罰, 當一條訊息表情符號超過四個, 會被視作洗版  
  
 **\[3\] 禁止自我宣傳** 除了特殊允許地方的廣告頻道/允許的身份組以外, 在其他頻道任何形式的自我宣傳, 將會被靜音作為懲罰 亦不允許任何私人訊息的宣傳  
  
 **\[4\] 連結方面** 除了 Youtube, Discord File attachment/embeds, Roblox, Tenor\(及其他認可的GIF網址\), Google Drive/Doc/Excel/Powerpoint/Form\(或其他許可的Google文檔網址\)以外, 一律不允許, 即使是Shorten被縮減的連結也被禁止, 為了防範某些用戶惡意散播虛假/釣魚連結\(ip收集器\), 防止該人士任何個人資料被入侵 本群組保留更改連結系統的最終決定權  
  
 **\[5\] 禁止一切歧視** 包括: 種族,性別,膚色,宗教及地區的任何歧視, 儒經發現會被靜音一天作懲罰  
  
 **\[6\] 年齡層注意** 本群組屬於PG-13等級的伺服器, 13歲或以下用戶, 必須留意!在這個群組你可能會發現少數不適當內容, 防範兒童進一步誤入歧途, 加入本伺服器最低年齡為 13  
  
 **\[7\] 禁止公開個人資料** 若果你披露任何人的個人資料\(年齡,住宅地址,真實個人名字,密碼等等\)令當事人感到煩惱, 本群將會刪除該訊息並進行懲罰  
  
 **\[8\] 禁止一切備用/雙開帳號** alt帳號會嚴重影響本伺服器的體驗, 除了被管理員許可的測試用途之外, 若果發現一律踢出  
  
 **\[9\] 禁止假冒他人** 例如: 大頭帖,用戶名稱/tag, 會被踢出  
因為我們會很難辨認誰是虛假及真實的, 請務必不要做以上的行為  
  
 **\[10\] 禁止任何NSFW內容** 若果以任何形式表達NSFW內容\(例如: 表情符號,文字,圖標,GIF等等會被禁止並會被靜音  
  
 **\[11\] 禁止散播不實內容** 例如任何連結及文章, 或是對任何成員進行欺詐, 即令該成員感到不安/令到財物損失, 即會被本群組封鎖  
  
 **\[12\] 禁止在非緊急情況下ping任何人士及Role** 若果在非緊急的情況下無意@任何身份組及用戶, 會對個別人士造成極大困擾, 若果發生以上行為, 會被嚴重警告及懲罰   
  
**\[13\]禁止一切濫用行為** 對於任何成員/管理員的任何濫用行為, 請前往舉報頻道 若果管理員被舉報兩次或以上可能會被考慮停職/永久取消該人士的身份組   
  
**\[14\]切勿濫用本群組資源** 例如在客戶服務部的Ticket服務等等即屬本群組資源, 若果你在有意地浪費本群組資源及管理層人員的時間將會被懲罰, 請將資源留給真真正正的有需要用的人士   
  
**\[15\]高級人員的非活躍狀態** 若果在72小時內管理層人員不上線/不進行有關本伺服器的任何活動, 可能會被調查及停職, 直到該人員達到本群組活躍要求除外 每週五晚上8:30\(GTM +8\), 管理層將會對所有等級身份組的所有用戶進行每周清查, 合要求的人士可保留該身份組, 否則將會被面臨不了身份組的機會   
  
**\[16\]歡迎諮詢我們** 若果你對以上任何內容及伺服器的所有內容感到疑惑, 請到支援頻道進行開票即可詢問  
  
**\[17\]Discord官方用戶條款** 年齡未經13歲的用戶會被本群組封鎖直到達到該年齡為止  
 在本伺服器不遵守TOS的人士會被本群組舉報
